Blockchain Interface

What is Blockchain?

Blockchain is a decentralized, distributed ledger that records the provenance of a digital asset. It is maintained by a network of nodes without the need for a central authority.

Each block contains a list of transactions, a reference to the previous block, and a proof of work.

Transactions are signed with a private key and can be verified with the corresponding public key.

Miners compete to solve a cryptographic puzzle to add a new block to the chain and receive a reward.

Connect with other nodes to share information and maintain a consistent blockchain across the network.

Use the interface below to interact with the blockchain, generate key pairs, send transactions, connect with nodes, create a genesis block, request coins from the faucet, check balance, and start mining.
